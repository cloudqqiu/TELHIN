#real data set for TELHIN

"test data.txt" contains all the manual-annotated data.
Each line can be divided by tab characters ('\t') into four segments: serial number, entity mention, real corresponding entity and tweet information.
The entity mentions are ambiguous actor names and the real corresponding entities refer to the distinct entities disambiguated by IMDb.

In the folder "extra", there are 26 txt files named by actor entity mentions.
In each file, each line can be divided by tab characters ('\t') into two segments: serial number and tweet information.

The tweet information may contain all the raw data gotten via Twitter API.
